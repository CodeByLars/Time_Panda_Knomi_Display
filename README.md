English:

Knomi Clock Mod is a robust solution designed to display the real-time, synchronized clock and date from your web server (using PHP) onto a Panda Knomi Display.
Utilizing an external Cron Job, a patched GIF image is updated every minute.
This GIF is then fetched by a Wemos D1 Mini (ESP8266) and pushed to the Knomi Display via OTA (Over-The-Air), ensuring a perfectly stable and synchronized timepiece.
